Super Store Sales & Profitability Dashboard

 Project Overview:
This Power BI dashboard provides a comprehensive analysis of sales performance and profitability for a global retail "Super Store." The goal of this project was to transform raw transactional data into actionable business insights, identifying high-performing regions, categories, and seasonal trends.

 -Key Insights:
 
Revenue Drivers: Technology is the leading category by revenue, significantly outperforming Furniture and Office Supplies.
Regional Performance: The West region contributes the highest share of sales (32.07%), followed closely by the East.
Profitability: Despite high sales volume in certain states, the overall Profit Margin remains at 13%, suggesting opportunities for cost optimization in lower-performing provinces.
Seasonality: Monthly sales trends show significant fluctuations, with peak performance occurring in the latter half of the fiscal year.

 -Technical Toolkit:
 
Data Visualization: Power BI Desktop
Data Transformation: Power Query (ETL process for cleaning and data type formatting)
Analytical Calculations: DAX (Data Analysis Expressions)

-Key Metrics Created:

Total Orders: Distinct count of Order IDs.
Profit Margin %: Calculated measure to evaluate bottom-line efficiency.
Revenue by Category: Aggregated sales performance.

 -Dashboard Features:
 
Executive KPIs: High-level overview of Total Sales (2.20M), Profit (280.76K), and Profit Margin (13%).
Time-Series Analysis: A dynamic line chart tracking Monthly Sales Trends to identify seasonality.
Geospatial Mapping: An interactive map highlighting profit distribution across global territories.
Interactive Slicers: Users can filter the entire report by Order Date and Customer Segment (Consumer, Corporate, Home Office) for granular analysis.
Top N Analysis: Bar charts identifying the top-performing states by revenue.


- How to Use:
  
Download the .pbix file from this repository.
Open the file using Power BI Desktop.
Use the Slicers on the top right to filter by date range or customer segment to see how the metrics shift.
